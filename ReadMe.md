# Summery
赤ちゃんの泣き声を検知してホーミーを鳴らすことで、泣き止ませるシステムがあればいいと思いました。

ホーミーの音は参考文献[1]を元に生成しました。
録音して流す機能があってもいいかもしれません。

# Dependency
## Framework
- System.Runtime.Serialization

## Nuget
- Math.Net Numerics
